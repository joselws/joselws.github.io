About

The fist working version of this project (as of this writing) was finished on March 18th 2022.

This is a Django project that gets data from anuncioneon.es website. You can look at the homepage, which has information about all the available categories and subcategories, and also explore available ads on their corresponding sections. Each ad has a title, subtitle, small description, price, and an optional image.

There is a filter section in which you can search for specific ads that matches your query on title and description. You may additionally also filter by category.

The heart of this app is on the web scraper program, in which all the important logic and features of this project are contained: From the requests and HTML parsing, to multithreading the data.

Obstacles and how I faced them

For most of the data fetching, parsing, and scraping process, it was pretty straightforward progress and posed no difficulties.

At some point, I had to extract a specific substring located in an "a" HTML tag, in the "href" attribute. I thought hard how to solve it using normal string operations on Python, but figured the most direct and easiest way to solve the problem was by using Regular Expressions.

All the data fetching from the 20 different category pages was taking a lot of time to complete, so I learned how to speed up the process by using multithreading. I spent significant time learning about this new concept for me, and decided to implement this behavior with ThreadPoolExecutor() in Python's concurrent.futures module. This cut the time it took to fetch all the data by 3.
